#### Start with Service Quota

1. Access the interface [Service Quotas](https://console.aws.amazon.com/servicequotas/home)

   - Select **Dashboard**
   - View overview **Dashboard**
   - In the **Dashboard** interface, select **Modify dashboard cards**

![Service Quotas](/images/1/0001.png?featherlight=false&width=90pc)

2. The currently selected services appear on the right. If you have selected nine services, you must remove one before you can add another. For each service that you don't need on the dashboard, select **Remove**

   - Then select **Save**

![Service Quotas](/images/1/0002.png?featherlight=false&width=90pc)

3. Change successfully.

   - Then select **Modify dashboard cards**

![Service Quotas](/images/1/0003.png?featherlight=false&width=90pc)

4. To add a service to the dashboard, select it from **Choose services**

   - After selecting the service, select **Save**

![Service Quotas](/images/1/0004.png?featherlight=false&width=90pc)

5. Complete the Dashboard update.

![Service Quotas](/images/1/0005.png?featherlight=false&width=90pc)
